~ FIREBASE AUTH SERVICE ~

	Users
	    uid: The unique, permanent ID for the user across your entire project. This is the "gold standard" key to use when saving data in your database.
	    displayName: The user's full name (e.g., "Jane Doe").
	    email: The primary email address associated with the account. (optional for anonymous users)
	    photoURL: A link to the user’s profile picture.
	    emailVerified: A boolean (true/false) indicating if the user has clicked the link in a verification email.
	    isAnonymous: true if the user is using a "Guest" account.
	    metadata.creationTime: The exact date/time the account was first created.
	    metadata.lastSignInTime: The last time the user successfully authenticated.
	    providerId: (e.g., google.com, facebook.com, password)
	    refreshToken: Used to get a new access token when the current one expires.

	PhysicalDesigns/{physicalDesignId}/
		id: string unique ID
		name: string name 
		description: string 
		author: uId
		authorFullName: author's display Name
		createdAt: creation datetime
		lastUpdatedAt: last updated datetime
		tags: [] list of tags
		savedCount: positive int of number of times this has been saved in the marketplace
		purchaseCount: positive int of number of times this has been purchased
		priceAmount: int numerical value for the price in USD cents (2999 -> "$29.99") 
		marketplaceStatus: "INACTIVE"|"PUBLIC"|"PRIVATE"
		marketplaceThumbnailUrl: string HTTPS URL for marketplace cards 
		marketplaceFullImageUrl: string HTTPS URL for detail page image 
		stripeProductId:
		stripePriceId:
	DigitalDesigns/{digitalDesignId}/
		id: string unique ID
		name: string name 
		description: string 
		author: reference to user's uid
		authorFullName: author's display Name
		createdAt: creation datetime
		lastUpdatedAt: last updated datetime
		tags: [] list of tags
		savedCount: positive int of number of times this has been saved in the marketplace
		purchaseCount: positive int of number of times this has been purchased
		priceAmount: int numerical value for the price in USD cents (195 -> "$1.95") 
		marketplaceStatus: "INACTIVE"|"PUBLIC"|"PRIVATE"
